# CampusConnect Web Application

## Setup & Run

1. Unzip the application package
2. Open integrated terminal from the directory of `server.js`
3. Check `config.env` for DB connection string and secret key *(do not modify)*
4. Run the following commands:
```
   npm install
   npm run watch
```
5. Go to `http://localhost:8000`
6. Click **Login** on the Top Navigation Bar
7. Enter credentials below & start exploring!
